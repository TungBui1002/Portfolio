## 🚀 Hướng dẫn deploy GitHub Pages

1. Vào [https://github.com](https://github.com) → Tạo repo mới, ví dụ `portfolio`
2. Upload toàn bộ file: `index.html`, `style.css`, `script.js`, `contact.html`, v.v.
3. Vào tab **Settings > Pages**
   - Source: `main` branch
   - Folder: `/root`
4. Nhấn **Save**
